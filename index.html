THNK

The GDevelop Multiplayer Framework for everyone!

Why THNK?Get started

Create all kinds of multiplayer experiences!


Local

THNK games work in singleplayer by default!
Multiplayer on the local machine is possible as well with the split-screen adapter (coming soon!)

LAN

You can easily make LAN games! With the Geckos.io adapter, start a server on your machine and let other connect to it via your local IP

World-wide

Players across the world can connect and play your game together with the THNK Cloud (coming soon!) and the P2P adapter
